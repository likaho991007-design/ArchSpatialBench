# Data Availability

This public project page does not include benchmark images, model outputs, or private annotations.

ArchSpatialBench uses a two-part data strategy:

## Core Data

Core Data is the redistributable benchmark split. It is intended to include license-compatible architectural drawings, annotations, benchmark questions, and scoring code.

Core Data is not yet publicly released.

## Wild Data

Wild Data is an in-the-wild evaluation split based on published architectural drawings. ArchDaily-hosted images are not redistributed.

Potential Wild Data release materials may include:

- Source URLs
- Metadata
- Hash manifests
- Benchmark questions
- Project-created annotations
- Evaluation scripts

We will not scrape, bulk-download, or redistribute ArchDaily-hosted content without written permission.
